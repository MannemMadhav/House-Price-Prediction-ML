# House Price Prediction using Machine Learning

## Overview
This project predicts house prices using Machine Learning and the King County Housing Dataset.

## Features
- House Price Prediction
- Flask Web Application
- Machine Learning Model
- Data Visualization
- Model Accuracy Display
- Responsive User Interface

## Technologies Used
- Python
- Flask
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- HTML
- CSS

## Dataset
King County House Sales Dataset

## Machine Learning Model
Linear Regression
